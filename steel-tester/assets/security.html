Security

Security Guidance

Wizard Toolz includes features that can work with real wallets, real funds, and real automation. That makes clear user safety habits essential.

Treat Automation Wallets As Hot Wallets

If you import a key, paste a key, or fund a generated wallet for a feature, assume that wallet is operating as a hot wallet in a live automation environment.

Fund Only What You Intend To Use

Do not overfund automation wallets. Keep only the amount needed for the feature you are running, plus a sensible gas buffer where appropriate.

Double-Check Token Addresses

Always verify the mint / CA before funding, launching, buying, or selling. Similar names, similar tickers, and copycat addresses are common in Solana trading environments.

Separate Sensitive Wallets

Use dedicated wallets for automation. Avoid mixing your main treasury, personal holdings, or long-term storage wallets into short-term trading or launch flows.

Support-Assisted Services

For services that require support coordination, always verify timing, availability, and expected delivery before assuming instant execution.

Need Help?

If you are unsure how to use a feature safely, contact support at @wizard_toolz before funding or starting the flow.

Privacy

Privacy Policy

Wizard Toolz is built to collect the minimum practical information needed to operate the Telegram bot, process feature flows, support users, and keep the platform stable.

What We Collect

We may store Telegram account identifiers, feature settings, generated wallet addresses, order records, activity logs, and other setup data needed to run the platform.

  • Telegram user IDs and bot interaction state
  • Wallet addresses and tool configuration data
  • Payment matching records and order history
  • Error logs, status logs, and support-relevant operational data

How We Use It

We use this information to operate the bot, deliver requested services, prevent abuse, investigate issues, and improve platform reliability.

  • Run automations and restore feature state after refreshes or restarts
  • Match payments and confirm service execution
  • Investigate problems, bugs, or refund cases when needed
  • Support users and improve feature quality over time

Private Keys and Sensitive Data

Some tools require wallet access. Those flows should always be treated as hot-wallet flows. Users should only provide keys or fund wallets they intentionally want to use with automation.

  • Do not submit more funds than you are prepared to automate
  • Rotate or retire hot wallets when appropriate
  • Use caution with any private key shared into bot-controlled flows

Third-Party Services

Some features rely on outside providers, routing systems, RPC services, or social/media delivery partners. Those providers may have their own data handling practices.

Contact

If you have privacy questions, support issues, or want clarification about any feature flow, contact support at @wizard_toolz.
